教育版：

将以下代码粘贴至Replit Shell后回车

`git clone https://github.com/valetzx/vscodeonreplit && mv -b vscodeonreplit/* ./ && mv -b vscodeonreplit/.[^.]* ./ && rm -rf *~ && rm -rf vscodeonreplit`

当加载完 Loading Nix environment... 后点击绿色 ▶ Run
